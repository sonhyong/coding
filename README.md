# 텍스트 RPG 게임

직업 선택과 스킬 사용, 쿨타임이 존재하는 다양한 기술로 전략적인 전투를 즐길 수 있습니다. WASD로 이동하는 맵과 몬스터가 배치되어 있으며, 인벤토리와 퀘스트가 포함된 간단한 스토리를 제공합니다. 간단한 텍스트 UI가 추가되어 전투와 메뉴 화면이 조금 더 보기 좋게 표시됩니다.
## 실행 방법
1. Visual Studio에서 새 콘솔 애플리케이션 프로젝트를 생성한 뒤 `Program.cpp` 파일을 추가합니다.
2. C++17 이상으로 설정하고 빌드 후 실행합니다.
   - 또는 터미널에서 `g++ Program.cpp -std=c++17 -o rpg` 명령으로 컴파일할 수 있습니다.
3. 게임을 실행한 뒤 영웅의 이름을 입력하면 마을에서 모험을 시작합니다.

게임에서는 이동과 전투, 스킬과 방어 행동을 선택하며 인벤토리를 관리하고 휴식할 수 있습니다. 숲의 퀘스트를 완료한 뒤 화산의 드래곤을 쓰러뜨리면 게임이 종료됩니다.
